Pyanocktail Server
============
  This is a twisted plugin designed to transform a Raspberry Pi in a Pianocktail, like described by Boris Vian in his book "L'Ã©cume des jours".
  
  Here is the software, you just need now a piano, a midi keyboard, a Raspberry Pi and some electronic skills to build your own Pianocktail! 

Requirements:

  	*Python2
  	*Alsa
  	*pyalsa
  	*twisted
  	*SQLAlchemy
  	*Autobahn websocket Python
  	*i2c tools (smbus library)
  	*scilab (optional)
	*numpy
